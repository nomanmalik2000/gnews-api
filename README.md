# gnews-api
This repo is demonstrating the useage of GNEW API.

## Usage
Clone this repo on your system
```
git clone <repo-link>
```

Create .env file from .env.example
```
GNEWS_API_KEY=<YOUR-GNEWS-API-KEY>
PORT=<APP-PORT-DEFAULT-IS-4000>
```

Install npm dependencies
```
npm install
```

Start node server
```
npm run serve
```
